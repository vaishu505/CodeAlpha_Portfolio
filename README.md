# Personal Portfolio Website

A modern and responsive **Personal Portfolio Website** built using **HTML, CSS, and JavaScript** as part of the **CodeAlpha Frontend Development Internship**.

## Features

- Home section with introduction
- About Me section
- Education details
- Experience section
- Skills showcase
- Projects section
- Contact information
- Fully responsive design
- Smooth scrolling and animations
- Modern user interface

## Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)

## How to Run

1. Clone or download this repository.
2. Open the project folder.
3. Open `index.html` in your web browser or use Live Server in VS Code.
4. Explore the portfolio website.
